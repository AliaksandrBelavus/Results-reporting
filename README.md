# JS Unit Testing

## Prerequisites
1. NODE version >=8.9.1
2. Git

## Install
```
npm install
```

## Run tests
```
npm e2e
```
### will create a `report` folder with:
* `cucumber_report.html` output

## Run e2e:update to update wedriver

## Run e2e:generate-report to generate report manually (or if any tests have failed)
